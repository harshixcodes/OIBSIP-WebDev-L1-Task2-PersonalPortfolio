Personal Portfolio Website

📌 Overview

This is a modern, responsive personal portfolio website developed using HTML5 and CSS3. It serves as my digital portfolio, showcasing my skills, projects, and contact information in a clean and professional design.

The website is fully responsive and provides a smooth user experience across desktop and mobile devices.

---

🚀 Features

- Responsive navigation bar with smooth scrolling
- Professional hero section with profile image
- About Me section
- Skills section with technology cards
- Projects section showcasing portfolio projects
- Contact section with social links
- Responsive layout using Flexbox and CSS Grid
- Modern UI with a consistent purple color theme

---

🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- javascript(for enhancements)

 💻 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your preferred web browser.

No additional installation or dependencies are required.

 🎯 Project Objective

The objective of this project is to create a responsive personal portfolio website that highlights my skills, projects, and contact information while practicing HTML and CSS fundamentals.
